# Aplikasi Penghitung Huruf Vokal dan Konsonan

## Deskripsi
Aplikasi ini merupakan aplikasi desktop berbasis Java Swing yang dibuat untuk
menghitung jumlah huruf vokal, konsonan, dan jumlah kata dari sebuah teks.
Aplikasi ini dibuat sebagai tugas Ujian Akhir Praktikum mata kuliah
Pemrograman Lanjut.


## Fitur
- Input teks / kalimat
- Menghitung huruf vokal (A, I, U, E, O)
- Menghitung total huruf vokal
- Menghitung huruf konsonan
- Menghitung jumlah kata
- Menyimpan data ke tabel
- Update dan hapus data
- Pencarian data
- Penyimpanan data ke file CSV


## Teknologi
- Bahasa Pemrograman: Java
- GUI: Java Swing
- Penyimpanan Data: File CSV



## Cara Menjalankan
1. Buka project di IntelliJ IDEA
2. Jalankan file `VokalUAPDarkFinal.java`
3. Aplikasi akan tampil dalam bentuk window


## Catatan
Aplikasi ini dibuat untuk keperluan pembelajaran dan tugas kuliah.
